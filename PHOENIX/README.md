## Core Principles

### 1. Immutability

Once recorded, data cannot be changed. Every entry is SHA-256 hashed and chained to the previous entry. Modifying any historical record breaks the chain.

### 2. Independence

We have no affiliation with the entities we monitor. No conflicts of interest. No incentive to misrepresent.

### 3. Transparency

All methodology is documented. All sources are cited. Anyone can verify our hashes independently.

### 4. Permanence

No retractions. No deletions. No memory-holing. The record stands forever.
