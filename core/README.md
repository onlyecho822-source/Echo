# Echo Nexus Core

Central orchestration kernel for Echo Nexus.

## Overview

The Core module provides the central orchestrator that coordinates all engines, frameworks, products, and services.

## Components

- **Orchestrator** - Central coordination
- **Configuration** - System configuration
- **Logging** - Unified logging
- **Metrics** - Prometheus metrics

## Usage

```python
from echo_nexus import Orchestrator

nexus = Orchestrator()
nexus.start()
```
