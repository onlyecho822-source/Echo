# E.X.O.D.U.S. Zine Contributions

## Submission Guidelines

This directory accepts community submissions for inclusion in E.X.O.D.U.S. zine publications. All contributions become part of the permanent cultural archive and are encoded with Adinkra ciphers.

## Accepted Formats

### Poetry & Text
- Markdown (.md)
- Plain text (.txt)
- Maximum 2000 words per submission

### Visual Art
- SVG (preferred for glyph compatibility)
- PNG (minimum 300 DPI)
- JPEG (for photographs)
- Maximum 10MB per file

### Memes & Digital Art
- PNG or GIF
- Include source files if applicable
- Describe intended cultural context

### Code Art
- Generative art scripts (Python, JavaScript)
- Must be functional and documented
- Include example outputs

## Submission Process

1. Create a new file in this directory
2. Name format: `YYYY-MM-DD_title_author.ext`
3. Include metadata header (see template below)
4. Submissions are reviewed for resonance score

## Metadata Template

```markdown
---
title: "Your Submission Title"
author: "Your Name or Pseudonym"
date: 2025-11-19
cipher: Sankofa
themes:
  - memory
  - sovereignty
  - resistance
resonance_intent: "Brief description of cultural intent"

[Your content here]
```

## Resonance Threshold

Submissions must achieve a resonance score of 6/10 or higher for publication. Score is calculated based on:

- Cultural alignment with E.X.O.D.U.S. principles
- Thematic coherence
- Technical quality
- Community relevance

## Content Guidelines

### Encouraged
- Counter-narratives to colonial history
- Ancestral memory and cultural reclamation
- Economic sovereignty and community wealth
- Technological sovereignty
- Healing and restoration themes
- Truth packets and historical corrections

### Not Accepted
- Content that reinforces colonial narratives
- Individual profit motives over community benefit
- Surveillance-enabling technologies
- Content that harms community safety
