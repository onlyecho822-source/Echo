# Echo Universe Books

This section contains long-form publications, manuscripts, and book-length documents produced by or for the Echo Universe project.

## Purpose

The books section serves as the repository for:

- **Manuscripts**: Draft and final versions of book-length works
- **Publications**: Completed books and monographs
- **Compilations**: Curated collections of research and analysis
- **Treatises**: In-depth explorations of specific topics

## Structure

```
docs/books/
├── manuscripts/       # Works in progress
├── publications/      # Completed and published works
├── compilations/      # Curated collections
└── treatises/         # Deep-dive documents
```

## Contribution Guidelines

1. All book-length content should be placed in this section
2. Use Markdown format for maximum compatibility
3. Include a front matter section with title, author, date, and abstract
4. Maintain version history in the document or via Git commits
