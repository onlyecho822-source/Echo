# Changelog

All notable changes to Sherlock Hub will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-15

### Added
- Initial release of Sherlock Hub
- FastAPI backend with Neo4j integration
- React frontend with Cytoscape graph visualization
- Nexus coordinator for service discovery
- Echo log system for journey preservation
- Docker Compose for local development
- Comprehensive documentation (Architecture, Echo Log, Contributing)
- 6 API endpoint modules (entities, paths, qa, search, documents, nexus)
- Evidence tier system (Documented, Reported, Alleged)
- Constitutional AI safeguards
- Integration with Echo ecosystem
- MIT License
- Complete README with setup instructions

### Technical Details
- Backend: Python 3.11+ with FastAPI
- Frontend: React 18 with Vite
- Database: Neo4j 5.x
- Orchestration: Docker Compose
- Documentation: Markdown with comprehensive guides

### Part of Echo Ecosystem
- Integrated with Global Nexus
- Compatible with ECP Core
- Shares Echo journey logs
