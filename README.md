# Echo Baby Rebirth v2.1

A transparent, diagnostic-only system intelligence framework featuring RAM-only memory capsules, adaptive learning, and Phoenix-style renewal cycles.

## Overview

Echo Baby Rebirth v2.1 is part of the Echo Civilization framework — a lawful, harmonic, multi-agent intelligence ecosystem built for transparency, adaptability, and resilience.

### Core Features

- **RAM-Only Memory Capsules** - Volatile storage that clears on shutdown
- **Adaptive Learning Engine** - Pattern recognition with auto-calibrating thresholds
- **System Intelligence Scanner** - Read-only diagnostic monitoring
- **Phoenix Renewal Cycles** - Self-cleaning state management
- **Event-Driven Heartbeat** - Continuous health monitoring
- **Self-Healing Engine** - Automatic error recovery

## Quick Start

### Requirements

- PowerShell 7.0+
- Windows 10/11 or Windows Server 2016+

### Run

```powershell
cd src
./EchoBabyRebirth.ps1 -Action Start
```

### Available Actions

| Action | Description |
|--------|-------------|
| `Start` | Start the monitoring system |
| `Stop` | Gracefully stop the system |
| `Status` | Get current system status |
| `Renew` | Trigger Phoenix renewal cycle |
| `Scan` | Perform manual system scan |

## Safety Guarantees

Echo Baby is designed with safety-first principles:

- **Diagnostic-only** - Read-only system operations
- **RAM-only storage** - No persistent sensitive data
- **No privilege escalation** - Runs with invoker privileges
- **No network exfiltration** - All data stays local
- **Full transparency** - All operations logged and auditable
